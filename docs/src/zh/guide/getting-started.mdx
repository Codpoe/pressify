# 快速上手

1. 新建目录

```shell
mkdir pressify-app
cd pressify-app
```

2. 初始化项目

```shell
yarn init
```

3. 安装依赖

<div className="text-red-500 pt-10 text-[31px]">123</div>

```shell
yarn add -D pressify
```

4. 在 `package.json` 中添加 scripts

```json
{
  "scripts": {
    "dev": "pressify dev",
    "build": "pressify build pressify build pressify build pressify build pressify build pressify build pressify build"
  }
}
```

5. 开始写文档

```shell
echo '# Hello Pressify' > README.md
```

6. 启动开发服务器

```shell
yarn dev
```

这会在 `http://localhost:3000` 启动一个开发服务器，打开这个地址应该就能看到你的站点了。
