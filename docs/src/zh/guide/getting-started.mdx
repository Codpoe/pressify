# 快速上手

1. 新建目录

```shell
mkdir pressify-app
cd pressify-app
```

2. 初始化项目

```shell
yarn init
```

3. 安装依赖

```shell
yarn add -D pressify
```

4. 在 `package.json` 中添加 scripts

```json
{
  "scripts": {
    "dev": "pressify dev",
    "build": "pressify build",
    "serve": "pressify serve"
  }
}
```

5. 开始写文档

```shell
echo '# Hello Pressify' > README.md
```

6. 启动开发服务器

```shell
yarn dev
```

这会在 `http://localhost:3000` 启动一个开发服务器，打开这个地址应该就能看到你的站点了。

7. 构建及部署

执行 `yarn build` 后，Pressify 默认会把构建产物输出到 `dist` 目录下，
我们可以将 `dist` 目录部署在 now.sh、GitHub Pages 等静态站点托管平台或者服务器上。
