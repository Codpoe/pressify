### 类似 VitePress，容易上手

```shell
# 在你的项目中安装
yarn add -D pressify

# 在 docs 目录下新建一个 markdown 文件
mkdir docs && cd docs
echo '# Hello Pressify' > README.md

# 开始写作
yarn pressify dev

# 构建静态文件
yarn pressify build
```
